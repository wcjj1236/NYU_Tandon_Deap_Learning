## Kaggle Notebook

The public Kaggle notebook associated with this project is provided by notebook-midterm.ipynb

## Reproducibility

This project is organized to support reproducibility of the submission pipeline:

- Use the **Colab notebook** to reproduce the main workflow.
- Use the **Kaggle notebook** to reproduce the competition-side inference environment.
- Use the **Google Drive resources** to access the required external model and data files.

## Usage Instructions

### 1. Open the notebook
Open the Colab notebook using the link above.

### 2. Download or mount external files
Access the required model and data files from the provided Google Drive links.

### 3. Update file paths if needed
Before running the notebook, update the file paths so that they match your own Google Drive, local environment, or Kaggle environment.

### 4. Run the pipeline
Run the notebook end-to-end to reproduce the inference workflow and generate `submission.csv`.

## Notes

- The main notebook is shared through Google Colab instead of being stored directly in this repository.
- The Kaggle notebook is included for public competition reproducibility.
- External resources are hosted on Google Drive because of file size limits.
- Path modification may be required before execution, depending on the runtime environment.
